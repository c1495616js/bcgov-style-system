### Fontawesome

- import `css` file in `typography.js`, and add `'Font Awesome 5 Free'` in `bodyFontFamily`.

```bash
npm i @fortawesome/fontawesome-free
```

```js
import '@fortawesome/fontawesome-free/css/all.min.css';
```
